Documentación del desarrollo de un prototipo funcional que digitaliza la experiencia de compra. El cliente interactúa con la app, se prueba la prenda en AR (Face Tracking) y accede a la web de pago.

PIZHUN es una marca ficticia de Streetwear. El producto es la PIZHUN Neural Cap. El objetivo es que el cliente experimente el producto mediante un Virtual Try-On y cierre la transacción en la web.

02. Ingeniería de Prompts 2D (Grok AI)

Utilizamos Grok Imagine para sintetizar las fotografías base de las prendas.

Prompt Base: "A hyper-realistic frontal product shot of a futuristic tech-core streetwear cap, matte black waterproof fabric, tactical buckles, volumetric studio lighting, 8K resolution, highly detailed."
⚠ Control de Físicas en Vídeo
Al animar fondos en Grok, la IA derretía las costuras. Lo solucioné añadiendo "perfectly static subject, ONLY handheld camera movement". Aislando el movimiento a la lente virtual, la ropa mantuvo su geometría.

03. Topología 3D y PBR (Meshy.ai)

La transformación espacial se delegó a Meshy.ai (Image-to-3D).

  1. Subida de la foto frontal de Grok.
  2. Exportación en High Quality (PBR Textures) para obtener mapas de Color, Normales y Rugosidad (necesarios para la iluminación en Unity).
  3. Exportado en .glb para comprimir geometría y texturas.
  4. Escala Crítica: En Unity, el modelo se escaló a 0.05 en X,Y,Z para adaptarse a la cabeza humana.

04. Desarrollo Front-End (DAW)

Desarrollada en Antigravity IDE, la pasarela de pago es Mobile-First.

  • CSS Grid: Se usó grid-template-columns: repeat(auto-fit, minmax(320px, 1fr)) para un catálogo responsive sin Media Queries densas.
  • Popups Nativos: Se utilizó la etiqueta HTML5 <dialog> que se superpone nativamente bloqueando el fondo (Top-Layer UX).

05. Desarrollo Unity: Interfaces y Face Tracking

Evolucionamos la rúbrica base (Image Tracking + BoxCollider) a un Try-On Facial con UI Canvas para mejorar la usabilidad.

5.1 Jerarquía y Flujo UI

El flujo consta de un Menú inicial, un Pop-Up y la Escena AR con un Canvas superpuesto que contiene botones estáticos (Volver y Comprar) en lugar de tocar el modelo 3D en movimiento.

5.2 ARHatController.cs (Offset Anatómico)

AR Face Manager ancla la malla en el centro del cráneo. Este script sube la gorra a la frente.

// Script adjunto al Prefab junto al componente ARFace using UnityEngine; using UnityEngine.XR.ARFoundation; [RequireComponent(typeof(ARFace))] public class ARHatController : MonoBehaviour { private ARFace _arFace; private GameObject _currentHatInstance; void Awake() { _arFace = GetComponent<ARFace>(); } void OnEnable() { _arFace.updated += OnFaceUpdated; } void OnDisable() { _arFace.updated -= OnFaceUpdated; } private void OnFaceUpdated(ARFaceUpdatedEventArgs args) { if (_currentHatInstance != null) { if (_arFace.trackingState == UnityEngine.XR.ARSubsystems.TrackingState.Tracking) _currentHatInstance.SetActive(true); else _currentHatInstance.SetActive(false); } } public void SetHatPrefab(GameObject newHatPrefab) { if (_currentHatInstance != null) Destroy(_currentHatInstance); if (newHatPrefab != null) { _currentHatInstance = Instantiate(newHatPrefab); _currentHatInstance.transform.SetParent(transform, false); _currentHatInstance.transform.localScale = Vector3.one; // OFFSET: Sube 15cm (Y) y adelanta 5cm (Z) hacia la frente. _currentHatInstance.transform.localPosition = new Vector3(0, 0.15f, 0.05f); _currentHatInstance.transform.localRotation = Quaternion.identity; } } }

5.4 HologramGlitchAnimator.cs (Estética UI)

using System.Collections; using UnityEngine; using UnityEngine.UI; [RequireComponent(typeof(CanvasGroup))] public class HologramGlitchAnimator : MonoBehaviour { private CanvasGroup _canvasGroup; private RectTransform _rectTransform; public float transitionDuration = 0.4f; private Vector3 _originalScale; private void Awake() { _canvasGroup = GetComponent<CanvasGroup>(); _rectTransform = GetComponent<RectTransform>(); _originalScale = Vector3.one; } private void OnEnable() { StartCoroutine(GlitchInCoroutine()); } private IEnumerator GlitchInCoroutine() { float elapsed = 0f; _canvasGroup.alpha = 0.2f; while (elapsed < transitionDuration) { elapsed += Time.unscaledDeltaTime; float t = elapsed / transitionDuration; _canvasGroup.alpha = Random.value > t ? Random.Range(0.1f, 0.8f) : Mathf.Lerp(0.5f, 1f, t); yield return null; } _canvasGroup.alpha = 1f; _rectTransform.localScale = _originalScale; } }
